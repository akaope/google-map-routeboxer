Oke, langsung saja. Sebelum mencoba source code ini sebelumnya buat dulu database dengan nama 'mapdb' (tanpa tanda petik)
setelah itu silahkan restore file mapdb.sql ke database tadi.

Setelah berhasil, di situ ada 3 tabel. Fokus saja ke tabel info_lokasi
di tabel itulah data-data lokasi yang digunakan, tapi fokus ke kolom latitude dan longitude saja karena memang kedua itulah yg paling penting

setelah itu coba akses localhost/local/route.php di browser ente

jika berhasil akan tampil map-nya. (INGAT! wajib tersambung ke jaringan internet)

sekarang coba klik tombol 'kirim'. Jangan ubah inputan 'surabaya' dan 'sidoarjo'-nya dulu
karena di database cuma ada data lokasi sekitar daerah surabaya saja.

Jika ingin mencoba lokasi lainnya, misal di sekitar daerah ente. Cara mendapatkan data latitude dan longitude
buka situs http://www.latlong.net/. masukkan latitude dan longitude ke database.

Selanjutnya silahkan ubah rutenya sesuai keinginan ente.
